# [hub_system_shorts] YouTube Shorts Script Patterns for Dev Tools
**Date**: 2026-05-19  **Type**: D1 (applied patterns)

## Validated Shorts Formula (from research)
- Length: 30-45 seconds
- Hook in first 3 seconds (make claim, show result, or ask sharp question)
- Text overlay on every scene (accessibility + retention)
- One clear CTA at end (pip install / GitHub link)

## Hook Types for Developer Audience
| Type | Example | Tension |
|------|---------|---------|
| Before/After | "Before: Claude forgets everything. After: 449 tasks tracked." | Loss → gain |
| Metric shock | "I made Claude 350x faster. Here's how." | Curiosity gap |
| Contrarian | "Stop using CLAUDE.md files." | Pattern interrupt |
| Show don't tell | Demo the UI in first 2 seconds, explain after | Visual proof |

## Structure for Hub-System Shorts
1. HOOK (0-3s): Bold claim with number
2. PROBLEM (3-10s): Claude loses context / no visibility
3. SOLUTION (10-25s): hub-system UI demo — North Star → milestones → stones
4. PROOF (25-40s): SQLite speed, Turso sync, 11 projects
5. CTA (40-50s): pip install northstar-hub
