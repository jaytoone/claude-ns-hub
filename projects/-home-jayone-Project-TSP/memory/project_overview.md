## 프로젝트: TSP (TrendSearch x Sales x Product)

**목표**: 트렌드 탐색(TrendSearch) → 세일즈(Sales) → 제품(Product) 전 과정(A to Z)을 설계하고 실험하는 프로젝트

**핵심 플로우**:
```
TrendSearch → Sales → Product
    ↑                     ↓
    └─────── Feedback ────┘
```

**Why:** 트렌드 신호를 포착해 세일즈 기회로 전환하고, 이를 제품화하는 전체 파이프라인을 검증하기 위함

**How to apply:** 각 기능 구현 시 이 세 도메인 간의 연결고리를 항상 고려할 것
