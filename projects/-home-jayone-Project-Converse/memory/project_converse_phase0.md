Converse Phase 0 measurement infrastructure designed and documented on 2026-04-03.

**Why:** /harness-live analysis revealed the core insight: "AI communication evolution is a measurement problem, not a content production problem." Three independent L2 agents (growth-loop, AARRR, research) converged on this. Without proper measurement (CS formula, silent failure detection), any deployment is blind.

**How to apply:**
- CS formula is FROZEN (reply*1.0 + unique_repliers*2.0 + depth*1.5 + return_visits*3.0, upvotes=0)
- Week 1-2 = baseline collection only (no AI content yet)
- Anti-scaling ceiling: cap AI participation at 10% per community
- Three failure pathogens to watch: Competent Stranger, Retrograde Normalizer, Ghost Town Effect
- Key deliverables in docs/phase0/: community-selection-criteria.md, measurement-framework.md, experiment-design-v1.md
