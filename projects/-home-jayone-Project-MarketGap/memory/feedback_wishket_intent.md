위시켓은 수주 채널이 아니라 시장가치 측정 + 성장 방향 탐지 도구로 사용.
인바운드 요청에 의도적으로 미지원할 수 있음 — 이는 정상 행동.

**Why:** 유저의 진짜 목적은 (1) 인바운드 빈도로 시장가치를 정량화하고, (2) 인바운드가 요구하는 스펙에서 자신의 역량 gap을 발견하고, (3) 그 gap을 실제 학습으로 줄이는 것. 수주/계약은 목표가 아님.

**How to apply:**
- 위시켓 관련 작업 시 "수주 전환율", "계약 성공" 등을 KPI로 제안하지 말 것
- 인바운드 빈도, 프로필 조회수, 지원서 열람률이 실제 KPI
- 인바운드에 미응답해도 "놓친 기회"로 프레이밍하지 말 것
- **프로필에 허위 기재 절대 금지** — 실제 학습 완료 + 산출물 존재 시에만 반영
- 인바운드 요구 스펙 → gap 분석 → 학습 로드맵 → 학습 완료 → 프로필 반영 순서 준수
- 프로필 변경은 A/B 실험으로 취급 (한 번에 1개 변수만)
