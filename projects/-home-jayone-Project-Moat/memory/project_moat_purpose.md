AI 분야 개인 MOAT(경쟁 우위) 구축 프로젝트. 자체 개발한 AI 기술(DARWIN V5 등)을 YouTube 콘텐츠로 전환해 기술 영향력과 수익 창출을 동시에 노린다.

**Why:** 커리어 자산화 — AI 엔지니어링 경험을 공개 콘텐츠로 전환해 개인 브랜드와 향후 수익(강의/컨설팅) 기반 확보

**How to apply:** 기술 분석 시 항상 "YouTube 콘텐츠 앵글"로 프레이밍. 일반 시청자에게 설명 가능한 비유와 시각화 중심으로 접근.
